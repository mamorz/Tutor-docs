\begin{frame}{Aufgabe: Asteroids}
    Folgende Klassen sind zu entwerfen:

    \begin{block}{Spieler}
        Ein Spieler mit Anzahl an Leben, einer Position mit x und y Koordinate als Ganzzahl, einer Rotation und einer Geschwindigkeit als Gleitkommazahl
    \end{block}

    \begin{block}{Asteroid}
        Ein Asteroid mit Größe und Geschwindigkeit als Gleitkommazahl und einer Position mit x und y Koordinate als Ganzzahl
    \end{block}

    \begin{block}{Spiel}
        Das Spiel enthält einen Score als Ganzzahl. Außerdem enthält es immer einen Asteroiden und einen Spieler
    \end{block}

    Entwerfen Sie die Klassen und kompilieren sie diese. Beheben sie eventuelle Syntaxfehler.
\end{frame}
