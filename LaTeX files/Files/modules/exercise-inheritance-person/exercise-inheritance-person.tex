\section{Aufgabe: Vererbung}

\begin{frame}{Aufgabe: Vererbung}
    Implementieren Sie folgende Klassenstrukturen möglichst sinnvoll.

    \begin{block}{Person}
        \begin{itemize}
            \item Eine Person hat einen Namen, ein Alter und eine Adresse
            \item Sie hat außerdem folgende Methoden \texttt{eat()} \& \texttt{sleep()}
            \item Ein Student ist eine Person mit einer Matrikelnummer und kann studieren
            \item Ein Dozent ist eine Person, hält eine Vorlesung und hat eine Personalnummer
        \end{itemize}
    \end{block}

    \begin{block}{Adresse}
        \begin{itemize}
            \item Eine Adresse ist entweder eine Postanschrift oder ein Postfach
            \item Ein Postanschrift hat Straße, Hausnummer, PLZ, Ort und Land
            \item Ein Postfach hat Nummer, PLZ, Ort, Land
        \end{itemize}
    \end{block}
\end{frame}
