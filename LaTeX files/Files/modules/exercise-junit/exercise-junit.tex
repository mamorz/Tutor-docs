\section{Aufgaben: JUnit}

\begin{frame}{Aufgaben: JUnit}
    \begin{block}{Aufgabe 1}
        Schreiben Sie eine JUnit Testklasse für die Primes-Klasse. Diese soll enthalten:

        \begin{itemize}
            \item mindestens einen Test
            \item mindestens eine JUnit Assertion
        \end{itemize}

        Ziel: 100 \% Testüberdeckung
    \end{block}
\end{frame}

\begin{frame}{Aufgaben: JUnit}
    \begin{block}{Aufgabe 2}
        Schreiben Sie eine JUnit Testklasse für eine Vektor-Klasse. Diese soll enthalten:

        \begin{itemize}
            \item eine Methode setUp() zur Initialisierung des zu testenden Objekts
            \item mindestens einen Test für jede Methode der Klasse
            \item mindestens einen Test, der eine Exception erwartet
            \item mindestens eine JUnit Assertion
        \end{itemize}

        Ziel: 100 \% Testüberdeckung
    \end{block}
\end{frame}
